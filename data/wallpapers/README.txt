JPEG 对比测试 — 测试图像数据集
====================================

来源: https://4kwallpapers.com/
总数: 428 张
分辨率: 3840×2160 (4K)
格式: JPEG / PNG
总大小: ~968 MB (压缩后 ~887 MB)

图像内容覆盖:
- 自然风景（山水、星空、森林、海洋）
- 汽车（保时捷、兰博基尼、法拉利等）
- 游戏场景（FPS、RPG、格斗等）
- 动漫/电影角色
- 抽象艺术、渐变背景

解压方法:
  Windows: 右键 → 全部解压缩，或使用 7-Zip/WinRAR
  Linux/macOS: unzip wallpapers.zip -d ../data/wallpapers/

下载脚本:
  如需更新数据集，请运行:
  python src/download_wallpapers.py --pages 20 --output data/wallpapers
